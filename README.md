# AI Mental Health App 🧠📱

Multimodal AI system for predicting student stress using:
- DNN (tabular data)
- LSTM (time-series)
- BERT (text analysis)

## Features
- Stress prediction
- Risk classification
- Mobile API integration

## Tech Stack
- PyTorch
- FastAPI
- Transformers

## How to Run
pip install -r requirements.txt
uvicorn src.api.main:app --reload
